HELP
Zurck zum Hilfe-Index 
zeigt eine ausfhrliche Hilfe zu einem DOS-Befehl an.

Syntax 

HELP [Name des gesuchten Befehles] [/o]

/o   ruft anstelle der New-DOS-Hilfe die Original-DOS-Hilfe auf. Dieser
     Parameter ist nur dann erforderlich, wenn das gesuchte Hilfethema sowohl
     in der New-DOS- als auch in der Original-DOS-Hilfe enthalten ist und Sie
     die Originalhilfe sehen wollen.

Wird kein Name oder der Parameter /? angegeben, zeigt HELP eine šbersicht.
Anwendung

HELP zeigt eine ausfhrliche Hilfe zu jedem DOS-Befehl. Das New-DOS-Programm
BROWSER.EXE muá dazu im New-DOS-Verzeichnis verfgbar sein. Das Hilfesystem 
Ihrer bisherigen DOS-Version wird von HELP eingebunden.
Beispiele

HELP copy
ruft die alte DOS-Hilfe auf und zeigt den Hilfetext zum Befehl COPY

HELP xcopy
ruft die neue DOS-Hilfe auf und zeigt den Hilfetext zu XCOPY

Ihre alte DOS-Hilfe wird also immer dann aufgerufen, wenn das angegebene 
Hilfethema in der New-DOS-Hilfe nicht verfgbar ist (oder HELP mit dem
Parameter /o aufgerufen wurde). Damit die alte Hilfe aufgerufen werden kann,
muá deren Pfadname bekannt sein - sollte der Aufruf der alten Hilfe also 
scheitern, mssen Sie diesen im Konfigurations-Dialog des Browsers
angeben.
