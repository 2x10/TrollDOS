Bestellung der Vollversion von NEW-DOS
Zurück zum Hilfe-Index
Wenn NEW-DOS-Programme die Meldung ausgeben, daß die Testzeit abgelaufen ist, dann besitzen Sie die Test-Version von NEW-DOS und sollten möglichst umgehend die ungesperrte Version beim Programmautor bestellen. Die Preisliste sowie ein Online-Bestellformular und weiterführende Informationen finden Sie im Internet unter http://www.newdos.de.

Verwenden Sie zur Bestellung ausschließlich die o.g. Internetseite.

Vorteile der ungesperrten Version von NEWDOS

  • Sie erhalten automatisch die allerneueste Version von NEW-DOS.
  • Im Internet liegt immer die aktuellste Version bereit, kleinere Updates
      werden kostenlos dort zur Verfügung stehen
  • unbegrenzte Laufzeit
  • Sie bekommen bei Problemen und Fragen Hilfestellung vom Programmierer von
      NEW-DOS, also aus allererster Hand
  • über Updates/Neuerungen werden Sie automatisch informiert
  • durch Wegfall der Sperrfunktion sind alle Programme kleiner und schneller
